经验

  • 千万不要以为多选就是多选，相信自己。
  • 做题是为了巩固基础，无论正确和错误，都要 看看题目的别人总结的解释，别为了做题而做题。
  • 这些题目都很基础，如果做的很好也没必要沾沾自喜，做几套公司的题目就知道了，做的不好也没必要难受，很正常，坚持下来就行，先定个小目标先来500道。
  • 没必要每天刻意去做题，每天抽出1，2个小时就可以。
  • 做完之后有时间的话可以在尝试做一遍，第二遍会非常快的。
